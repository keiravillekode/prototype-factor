# Learning

- [Official documentation][docs] -- the complete reference for all built-in vocabularies and words. Searchable from the GUI Listener or the command line.
- [Factor cookbook][cookbook] -- getting started guide covering key concepts.
- [Your first program][first-program] -- the official walkthrough for writing, testing, and documenting a vocabulary.
- [Factor tutorial][tutorial] by Andrea Ferretti -- a comprehensive introduction covering the stack, combinators, vocabularies, and more.
- [Learn Factor in Y Minutes][y-minutes] -- a quick tour of Factor syntax and features.
- [Factor on Rosetta Code][rosetta] -- solutions to many programming problems in Factor, useful for learning by example.
- [Factor GitHub Wiki][wiki-gh] -- community-maintained wiki with tips and guides.
- [Concatenative wiki: Factor][wiki] -- community wiki with tutorials, examples, and FAQ.
- [Factor blog][blog] -- articles and announcements from the Factor developers.
